To add a submodule:
git submodule add <URL> <PATH>

To initialize a submodule:
git submodule update --init --recursive

To update a submodule:
git submodule update --remote


1. Preprocessing: .c, .h -> .i
   Resolve macros and directives.
2. Compilation: .i -> .s
   Translate preprocessed source code into assembly code.
3. Assembly: .s -> .o
   Convert assembly code into machine code.
4. Linking: .o -> single executable
   Combine object code into single executable.


Follow this:
https://files.portaudio.com/docs/v19-doxydocs/compile_windows_mingw-w64.html

(for x86_64)

Install MSYS2 to use g++ (and cmake) on Windows.

pacman -S mingw-w64-x86_64-gcc mingw-w64-x86_64-cmake mingw-w64-x86_64-portaudio

Quick note on MSYS2: MSYS2 comes with different environments (MSYS, UCRT64, MINGW64, etc.).
All environments inherit from MSYS and add additional things on top of it. Use MINGW64.

To build:
cmake -B build
cmake --build build
